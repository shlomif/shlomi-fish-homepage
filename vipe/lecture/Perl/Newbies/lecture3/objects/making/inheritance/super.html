4.3.3.1 Calling Over-riden Methods of Base Classes

It is possible to explicitly call the method of a base class even if it was over-riden by another method in the derived class. To do that use the SUPER:: prefix before the method name.

Here is a re-write of the Bar class, this time with assign_name retaining its name. 


package Bar2;

use strict;

use vars qw(@ISA);

use Foo;

@ISA=qw(Foo);

sub assign_name
{
    my $self = shift;

    my $name = shift;

    # Call the method of the base class
    my $ret = $self->SUPER::assign_name($name);
    if (! $ret)
    {
        $self->{'num_times'}++;
    }

    return $ret;
}

sub get_num_times_assigned
{
    my $self = shift;

    return
        (exists($self->{'num_times'}) ?
            $self->{'num_times'} :
            0
        );
}

1;
