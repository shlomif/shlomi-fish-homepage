9.2.1 The for keyword and the .. operator

The for keyword in Perl means exactly the same as foreach and you can use either one arbitrarily.

$a .. $b is a special operator that returns an array containg the sequence of consecutive integers from $a up to and including $b. Now one can fully understand the for $i (1 .. 10) construct that we used in the beginning of this lecture.
