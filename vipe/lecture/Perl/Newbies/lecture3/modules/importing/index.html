3.2 Importing Modules and their Functions

In order to have access to a module from within your script (or from within another module) you can use the use directive followed by the name of the module as it appeared in the package statement. Here's an example: assume that the file "MyModule.pm" is this: 

# This is the file MyModule.pm
#

package MyModule;

use strict;

sub a_function
{
    print "Hello, World!\n";
}

1;

And this is your script: 

#!/usr/bin/perl

use MyModule;

# Notice that we use "::" to call the function out of the module.
MyModule::a_function();

That way, the program will print "Hello, World!" on the screen.
