7.2 Script-Fu Scripting

  • Script-Fu enables to write GIMP Scripts and PDB entries in Scheme.
  • The Scheme scripts reside in ~/.gimp-1.2/scripts
  • Whenever changed they need to be refreshed by selecting    <Toolbox> -> Xtns -> Script-Fu -> Refresh
  • Besides that, GIMP sports an interactive Script-Fu Console.
