Gamla - a perl-based RAD and application server

Welcome to the homepage of the Gamla project. The gamla project aims to create a Rapid Application Development (RAD) tool and a web application server based on Perl. The two objectives were placed under the hood of one project because I (Shlomi Fish) feels that they are closely related.

All the source code produced by the Gamla project will be under the public domain. External perl modules that will be used (for example, DBI) will remain under their original license, assuming we find the need to modify them.

If you wish to help, or just participate in the discussions, then join the gamla-devel mailing list.
