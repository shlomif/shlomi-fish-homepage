Program to check if two graphs have the same topology

GraphIso.pm - This is the main module with the function that does the comparison.

Graphs_Equal.pl - a test program that uses the GraphIso module and runs on several test graphs. They can be generated by gen_graphs.pl below.

gen_graphs.pl - This is the script that generates some sample graphs for input to the program. It does so in a platform-independant way.

test.pl - This script runs Graphs_Equal.pl and gen_graphs.pl consecutively for a series of seeds. To run it type "echo [initial seed] > seed.txt".
