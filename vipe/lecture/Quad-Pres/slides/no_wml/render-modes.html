5.2 The Render Modes

  • QuaD-Pres without WML has three render modes:
  • Server renders the pages for inclusion on a web-server's static HTML directory.
  • Hard-Disk renders the pages for viewing on a local hard-disk.
  • CGI - it is possible to use a CGI script to dynmically render the pages.
  • "Server" and "Hard-Disk" modes can be specified in the target types inside Render_all_contents.pl
