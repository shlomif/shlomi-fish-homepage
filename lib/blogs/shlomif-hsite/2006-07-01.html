Well, there were a few new additions to my home site since the last update, but not enough to warrant a new update. However, I'd like to take this stage to announce a URL transition and to also ask for people's advice regarding some issues.

However, one important note is that the sub-section of the site that used to live under http://vipe.technion.ac.il/~shlomif/ is now living under http://www.shlomifish.org/Vipe/.

This change was done due to the constant downtime of vipe.technion.ac.il. Note that the server itself is usually up and running and responsive, but the connection to it gets down. Another benefits of this are the fact that everything would be on one host, which would make mirroring, backup and maintenance easier.

A more long-term plan would be to move all resources directly under http://www.shlomifish.org/, but that will take some extra work, and will be done incrementally. Meanwhile, there may still be some links to the old domain. I hope to find them all by myself, but if you can still find some, please report them to me.
