4.3 An Individual Page

  • After you run Render_all_contents.pl initial templates for all the individual pages will be generated for you in their place under the src sub-directory.
  • The line #include 'template.wml' need to be kept at the beginning of the file.
  • Afterwards you can use whatever HTML and WebMetaLanguage code to fill its contents.
  • The meta-tag <qpcontents /> evaluates to a table of contents of the current node and inwards.
  • Example:
#include 'template.wml'
