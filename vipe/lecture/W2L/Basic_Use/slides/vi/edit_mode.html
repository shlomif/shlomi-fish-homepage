4.4 Entering and Exiting from Editing Mode

  • To enter editing mode, use the arrow keys to move the cursor to the location in which you wish to enter text, and press "i".
  • You can now enter text freely, and delete the text that was just entered by pressing the backspace key.
  • When you are done entering text, press ESC to return back to c ommand mode.
  • Note: Pressing ESC twice will always put you back in command mode.
