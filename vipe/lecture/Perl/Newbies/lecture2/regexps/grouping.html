7.4 Grouping with "( ... )"

One can use the parenthesis (( ... )) to group several characters so the entire group can be repeated with +, * or ?. For example, the following perl program sees if it's argument is an entire sentence that starts with "The" and ends with "there": 

use strict;

my $string = lc(shift(@ARGV));

if ($string =~ /the( +[a-z]+)* +there/)
{
    print "True\n";  
}
else
{
    print "False\n";
}

It is possible to nest groupings, so for example the following matches a semi-colon separated list of comma-separated lists of numbers (encolsed inside square brackets): 

use strict;

my $string = lc(shift(@ARGV));

if ($string =~ /\[([0-9]+(,[0-9]+)+)(;([0-9]+(,[0-9]+)+))+\]/)
{
    print "True\n";  
}
else
{
    print "False\n";
}
