2.5 Plug-ins

  • In the GIMP, effects and filters are implemented using plug-ins.
  • Plug-ins are separate programs that communicate with the main GIMP program using pipes.
  • Most of the plug-ins that are distributed with the GIMP are GPLed, but it is legal and possible to write proprietary plugins.
  • Besides plug-ins written in C, GIMP ships with Scheme and Perl Extensions, that enable writing filters and scripts in these languages.

Demo

  1. Create an image with a text caption.
  2. Invoke <Image> -> Filters -> Blur -> "Blur..."
  3. You can temper with the settings and then press OK.
  4. The image is now blurred.
