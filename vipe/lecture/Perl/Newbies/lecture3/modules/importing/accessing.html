3.2.1 Accessing Functions from a Different Module

As could be seen from the last example, once the module has been use'd, its functions can be invoked by typing the full path of the module followed by :: and followed by the function name.

Note that if you are in module "Foo" and you are trying to access functions from module "Foo::Bar", than typing "Bar::my_func()" will not do. You have to type the full path of the module.
