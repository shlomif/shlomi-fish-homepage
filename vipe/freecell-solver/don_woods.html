Don Woods' Freecell Solver

don_woods_solver.zip

This is the source code for the famous Freecell solver by Don Woods. It is basically 4 different solvers for four different Freecell variants which are using a very similar algorithm. Maybe it is possible to abstract the game rules and integrate all of them into one solver.
