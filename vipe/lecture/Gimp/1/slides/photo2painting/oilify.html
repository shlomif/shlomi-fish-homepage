6.1 Oilify

  • The "Oilify" tool enables one to turn a photograph (or something similar) into an oil painting-like image.

Demo

  1. Open g_anwar1.jpg
  2. Select    <Image> -> Filters -> Artistic -> "Oilify..."
  3. Selct a mask size and press the OK Button.
  4. Undo and Play with the various settings.

  • Oilify works by replacing the pixel at (x,y) with the value that occurs most often in its N x N region.
  • It does so separately for each RGB Channel.
