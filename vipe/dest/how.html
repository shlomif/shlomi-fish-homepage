How to Get it?

The project is being developed Using SourceForge's system. The project's page on sourceforge is http://www.sourceforge.net/projects/syscalltrack. You can find the lastest version to download on that page, as well as access to the CVS repository

  • Browse it on line.
  • Download the really up-to-date version from CVS.

Contact Info

For general inquiries about the syscall tracker, bug reports and complaints, please send email to syscalltrack-hackers@lists.sourceforge.net.
This mailing list is mostly for developers, but for now may be used for other inquiries as well now. If we will see that there is growing traffic on the mailing list, we will create a separate mailing list for syscalltrack users.

Information about the mailing list, how to register, and how to access the list's archives, can be found at http://lists.sourceforge.net/lists/listinfo/syscalltrack-hackers.
