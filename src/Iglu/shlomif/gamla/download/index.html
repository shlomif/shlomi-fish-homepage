Gamla Downloads

Shlomif::MiniReporter - a perl package that was used in the Jobs and Consultants databases of the Linux-IL website. Very basic functionality. To be replaced with Arad::OneTable::Reporter. Arad::OneTable::Editor - a set of perl modules to edit an SQL table. Useful, but has some very basic functionality. Still, it contains some important concepts. Arad::OneTable::RecordStream - Now this should be part of the core Gamla. A class for inputting a stream of records whether SQL or not. Arad::Types - Gamla's Type manager. Under development. Gamla::TextStream::Out - A nifty little class that outputs text to a buffer, a filehandle or whatever. Gamla::Web::PathMan - (Under development) A URL and path manager.
