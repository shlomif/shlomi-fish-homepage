Interesting Projects

syscalltrack has drawn the attention of various academy people.

  • An Msc student who deals with fualt tolerance
  • Roy Friedman from the computer science department in the technion is giving a project regarding the development of syscalltrack this very summer.

Some other projects which work on the same grounds as syscalltrack

  • SysTrace - systrace and syscalltrack aim for different goals. We want to do debugging and needle like precision, "which application is deleting that file when running under the user UID?", and systrace aims to do application level "jails". syscalltrack is written in such a way that once we have the infrastructure in place, we would develop the systrace capabilities on top of it. At that time, it might make sense to borrow and cooperate, in the best open source tradition, from systrace.
  • medusa - A security oriented project, which catches certain system calls, and either allows, denies or replaces them with others. Guy explains the difference.
  • overloader - a user space, script based tool, which alerts over invalid system calls.
