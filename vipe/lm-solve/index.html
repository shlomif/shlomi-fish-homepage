LM-Solve

LM-Solve is a set of solvers I wrote for the Logic Mazes featured on the Logic Mazes site. They are written in Perl and should be very portable.

Download LM-Solve Version 0.6.0

The Logic Mazes Applets themselves

Download Layouts for some of the puzzles on the site. - I just supply them, but I am not their legal owner. If you write down others, please send them to me.

News

15-Aug-2002

LM-Solve version 0.6.0 was released. This versions sports support for Planks in the Swamp Puzzles (rectangular only), and can be built as an RPM out of the box.

12-Jun-2002

LM-Solve version 0.4.0 was released. This version features support for three types of tilt mazes: single goal, multiple goals, and Red-Blue Tilt Puzzles
