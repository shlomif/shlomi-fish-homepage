9.1 grid

  • grid is an abstraction around the HTML table. 

#use wml::std::grid
<grid layout=2x3 align=lr valign=tbb border=1>
    <cell>Header 1</cell> <cell>Header 2</cell>
    <cell>Cell-A</cell>   <cell>Cell-B</cell>
    <cell>Cell-C</cell>   <cell>Cell-D</cell>
</grid>
